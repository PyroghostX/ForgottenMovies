Scheduler

When the scheduler is disabled, Forgotten Movies will skip automatic Seer/Tautulli scans and will not send reminder emails. Manual "Run Now" actions still work.

Run Now

Kick off the Forgotten Movies check immediately instead of waiting for the next schedule.

This checks Seer for new requests, checks Tautulli for unwatched old requests, and sends any eligible reminder emails.

Update Watch Status

Pressing this will check to see if people have watched what we have reminded them on.

This checks all unwatched sent emails against Tautulli to see if users have watched the content. This happens automatically during scheduled runs, but you can trigger it manually here.
